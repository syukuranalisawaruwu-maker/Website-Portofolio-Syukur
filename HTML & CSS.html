HTML & CSS

HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets) adalah fondasi utama untuk membuat website modern. Saya memahami kedua teknologi ini secara menyeluruh sehingga dapat membangun website yang rapi, menarik, dan responsif. Pengetahuan saya mencakup struktur halaman, elemen semantik, layout, desain visual, animasi, dan teknik responsive untuk berbagai perangkat.

  • HTML Dasar & Lanjutan: Mampu membuat struktur halaman dengan heading, paragraf, daftar, tabel, form input, link, gambar, audio, video, serta elemen HTML5 modern.
  • CSS Styling: Mengatur tampilan website dengan warna, font, background, border, shadow, dan efek visual.
  • Layout & Responsiveness: Menggunakan Flexbox dan CSS Grid serta media queries.
  • Animasi & Interaksi: Membuat animasi, transisi, hover effects.
  • Struktur & Praktik Rapi: Pengelolaan file yang baik dan scalable.
  • Proyek Nyata: Website portofolio, landing page, dan website sekolah.
  • Design Modern: Glassmorphism, gradient, rounded corner.
  • Integrasi JavaScript: Siap untuk interaktivitas lanjutan.

Dengan kemampuan ini, saya mampu membuat website dari awal hingga siap diintegrasikan dengan teknologi lain.
