Project Desain Grafis

Deskripsi Project

Project desain grafis ini berfokus pada pembuatan desain visual dan antarmuka aplikasi. Saya mendesain gambaran aplikasi pengelolaan sampah dengan tujuan menciptakan tampilan yang bersih, modern, dan mudah digunakan oleh pengguna.

Hasil Desain

Tools yang Digunakan

  • Figma untuk desain UI dan prototype
  • Color palette dan typography modern
  • Icon dan komponen UI

Apa yang Saya Kerjakan

  • Membuat konsep desain aplikasi pengelolaan sampah
  • Mendesain tampilan halaman aplikasi
  • Mengatur layout agar rapi dan konsisten
  • Membuat prototype interaktif menggunakan Figma
