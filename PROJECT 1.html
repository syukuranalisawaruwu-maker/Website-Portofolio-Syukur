Project Website Portofolio

1. Gambaran Umum Project

Website portofolio ini saya buat sebagai media untuk menampilkan identitas diri, keterampilan, serta hasil project yang pernah saya kerjakan. Website dirancang dengan tampilan modern, rapi, dan responsif agar mudah digunakan di berbagai perangkat.

2. Tujuan Pembuatan

Tujuan pembuatan website portofolio ini adalah sebagai sarana personal branding, dokumentasi hasil belajar, serta media untuk menunjukkan kemampuan saya dalam pengembangan website.

3. Proses dan Pekerjaan yang Dilakukan

  • Perencanaan Struktur: Menentukan halaman utama, skill, project, dan navigasi.
  • Desain Tampilan: Menggunakan konsep modern, gradient warna, dan layout rapi.
  • HTML & CSS: Membuat struktur halaman dan desain visual.
  • Responsive Design: Menggunakan Flexbox dan media queries.
  • Dark & Light Mode: Mode otomatis tersimpan di semua halaman.
  • Navigasi Interaktif: Scroll langsung ke bagian tertentu.

4. Hasil dan Manfaat

Dari project ini saya memperoleh pengalaman membangun website dari nol, meningkatkan kemampuan desain dan struktur web, serta menghasilkan portofolio yang dapat dikembangkan lebih lanjut.
